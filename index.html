Previous Projects

TGR Fitness

This is a fully responsive, custom-built website developed for a personal trainer. This project showcases a clean, modern design tailored to promote my clients personal training services. Special attention was given to responsive layouts, mobile optimization, and to convert visitors into clients.

Royal Mail Replica

A fully responsive front-end replica of the Royal Mail homepage, The project closely mimics the structure, layout, and visual style of the original site, with attention to responsive navigation, modern layout patterns, and mobile usability This project highlights my ability to interpret and reproduce complex UI components from scratch.

Arsenal Website Replica

This project is a fully responsive front-end replica of the Arsenal FC homepage. Built entirely from scratch, it was designed to closely mirror the original site's layout, and user interface. The goal was to practice real-world responsive design techniques while replicating a professional-grade website to showcase my skillset.
